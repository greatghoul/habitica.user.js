# Habitica User Scripts

Collection of Tampermonkey user scripts for [Habitica](https://habitica.com).

## Scripts

### Hide System Messages

Toggle system message visibility in party chat, hidden by default.

[INSTALL](https://github.com/greatghoul/habitica.user.js/raw/refs/heads/main/Hide_System_Messages/main.user.js) · [Source](./Hide_System_Messages)

### Upload Image

Upload image to chat via paste or toolbar button. Supports JPG, PNG, GIF.

[INSTALL](https://github.com/greatghoul/habitica.user.js/raw/refs/heads/main/Upload_Image/main.user.js) · [Source](./Upload_Image)

## Install

Make sure you have [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) or similar userscript manager installed, then click the install link above.
