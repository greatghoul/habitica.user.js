## Description

Upload image to Habitica chat via paste or toolbar button.

## Features

- Paste image from clipboard directly into chat input
- Click the image icon in the toolbar above the chat textarea to select a file
- Supports JPG, PNG, GIF formats
- Upload progress indicator with spinning icon

## Install

Make sure you have [Tamermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) Chrome Extension or similar addon installed.
